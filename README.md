Rock Paper Scissors Game (Python)

Description

This is a simple Rock Paper Scissors command-line game written in Python.
The player competes against the computer. The computer randomly selects a choice, and the user enters their own choice.

The program then compares the choices and determines the result.

Features

- Random computer choices using Python's "random" module
- Input validation to ensure correct user input
- Displays both the computer and player selections
- Allows the player to restart the game multiple times
- Simple command-line interface

Technologies Used

- Python
- Standard Python libraries:
  - "random"
  - "os"
  - "shutil"

How the Game Works

1. The computer randomly selects one of the following:
   
   - rock
   - paper
   - scissors

2. The player is asked to enter their choice.

3. The program compares the two choices and prints the result.

4. The player can choose whether to play again.
